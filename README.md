# 摸鱼神器

## 概述

本摸鱼神器适合用电脑工作时看小说，目前支持UTF-8和ANSI编码的文本。当导入文本文件后，会自动创建透明背景的文本窗口，同时提供多种自定义选项，让打工人摸鱼更放心。

## 已开发功能

自定义字体类型、大小、颜色和透明度

自定义显示的文本长度、行数、行间距

自定义快捷键翻页，默认```ctrl + c```与```ctrl + x```进行上下页翻页

记忆阅读位置，下次打开会跳转到上一次阅读的位置
